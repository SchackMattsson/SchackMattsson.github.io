Text indent and font variant


The text indent property specifies indentation of the first line in a text block. In this text-indent is set to 40px.

Lorem ipsum, dolor sit amet consectetur adipisicing elit. Minus nihil reiciendis impedit voluptatum
voluptas. Voluptatum nisi perferendis autem sunt soluta veniam blanditiis nostrum atque harum, odit, voluptatibus ad,
maiores architecto!

The font variant property specifies whether text will be displayed in a small-caps font. The font-variant in this text is set to small-caps.

Lorem ipsum, dolor sit amet consectetur adipisicing elit. Dolorum sunt, est minima eaque deserunt unde
magni laboriosam nostrum omnis, fuga odit libero mollitia consectetur sit dolorem veniam sequi, qui adipisci!'
